# Phase Calibration
This is a Desmos project to play music in Desmos

Simply run the python file to generate the short time fourier coefficients that you can paste into the data folder of this graph: https://www.desmos.com/calculator/tmqsxilkye
The Desmos side of this project works by calibrating the phases of each tone by exploiting the inner workings of the WebAudio API.
Now anyone can play quality audio on Desmos
